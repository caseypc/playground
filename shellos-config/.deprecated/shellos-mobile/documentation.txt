1. Partition internal SD card
    - get into ClockworkMod Recovery by holding down Volume Up + Home + Power buttons together
    - partition internal SD card (ext 4096M, swap 256M)
1. Install ClockworkMod Recovery
    - put your phone into download mode by holding down Volume Down + Home + Power buttons together
    - download ROM that includes ClockworkMod Recovery, SEE: http://wiki.cyanogenmod.com/wiki/Samsung_Galaxy_S_II:_Full_Update_Guide
    - upload ROM to the phone using Odin
    - remove Yellow Exclamation mark using USB Jig
2. Install CyanogenMod
    - download desired version of CyanogenMod, SEE: http://get.cm/?device=i9100&type=nightly
    - download desired version of Google Apps, SEE: http://wiki.cyanogenmod.com/wiki/Latest_Version#Google_Apps
    - remove assert statement from cyanogenmod-archive/META-INF/com/google/android/updater-script and re-pack the archive
    - copy CyanogenMod archive to the SD card
    - get into ClockworkMod Recovery by holding down Volume Up + Home + Power buttons together
    - install CyanogenMod and Google Apps
    - reboot
3. Configure
    - login to the Google account
    - configure device settings
    - install applications
4. Backup
    - create backup using Titanium Backup
