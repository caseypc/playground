This project provides a set of scripts to automate installation of Windows XP operating system along with optional programs. *Software requirements*
[http://www.open.collab.net/downloads/subversion/ Collabnet Subversion Client]SVN cilent
[http://www.7-zip.org/ 7-Zip]file archiver
[http://technet.microsoft.com/en-us/sysinternals/bb896768.aspx Junction]tool to create directory links
[http://www.nliteos.com/ nLite]Windows XP installation customizer
Windows XP x64Windows XP installation CD
*How to use it* #

Install required software and configure your environment.

#

Export scripts from the repository using Collabnet Subversion Client. You could use this command:svn export http://osbuild.googlecode.com/svn/trunk/ osbuild

#

Create a zip file _winxp.zip_ using 7-Zip with the content of your Windows XP x64 installation CD and place it in _winxp\windows_ directory in the project's directory structure.

#

Edit _"winxp\windows\Last Session_u.ini"_ file. At least you have to assign a valid serial number of your Windows XP copy to the _CDKey_ property.

#

Download programs of your interest from the list below and place them in _winxp\programs_ directory.

#

Edit _winxp\programs\install.cmd_ file. At least check if downloaded file names correspond to file names used by the script.

#

Run _winxp\build.cmd_ script.

#

Create a bootable ISO image using nLite with a content of _winxp\workspace_ directory.
